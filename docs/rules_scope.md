# Rules Scope (Schieber-first)

## Phase order (implement in this order)

1. **Schieber base game**
   - Modes: Trumpf (suit), Obeabe, Uneufe
   - Must-follow-suit, must-trump, must-overtrump (per Schieber rules)
   - Scoring: card points + last trick +5
   - Total points per round: 157 (without Stoeck/Weis)

2. **Stoeck**
   - Trumpf King + Trumpf Ober = 20 points
   - Timing/announcement rules per official Swiss Jass rules

3. **Weis (announcements)**
   - Schweizer Reglement (combinations, priority, announce/counter)
   - Keep as a separate, optional module

4. **Variants / tournament options (if needed)**
   - Rule-set flags for regional/tournament deviations
